Configuración

Se usa para el agente de indicadores y para herramientas cuando el proveedor las soporta. Se usa para patrones, tendencia y decisión final. Para Copilot, el valor recomendado es Claude Opus 4.6.
Usa la sesión local de Copilot CLI en esta máquina. También puedes guardar un token de GitHub explícito como override opcional.

Información de seguridad

Solo localhost

Este servidor se ejecuta exclusivamente en localhost (127.0.0.1)

Seguridad de la clave API

Tu clave API se almacena localmente y nunca se sube a servidores externos

Privacidad de los datos

Todos los datos del análisis permanecen en tu equipo local
